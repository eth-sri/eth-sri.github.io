--- title: Q&A Static taint analysis for Ethereum contracts ref: pass2020-project-qa --- This document contains common questions by students and answers by the TAs.


  • The Datalog template (analyze.dl) declares the input facts "structLoad" and "structStore". Do we need to use these?
    Answer: No. You can ignore structLoad and structStore as you do not need to support structuers.

  • The project description states that we must support the message variables such as msg.sender and msg.value. What other message variables should we support?
    Answer: You need to support only msg.sender and msg.value.

  • A store input facts is given by "S10 V00 S09", where S09 is stored into V00. What is assigned to variable S10?
    Answer: Nothing is stored at S10 because it is a store statement.
