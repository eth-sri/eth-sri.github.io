Overview

Security issues and privacy breaches in modern systems (blockchains, datacenters, AI, etc.) result in billions of losses due to hacks and system downtime. This course introduces state-of-the-art automated techniques and demonstrates how they are applied to build reliable and secure systems.

Objectives

  1. Understand how classic automated analysis and synthesis techniques work, including discrete and probabilistic methods.
  2. Understand how to apply these methods to either attack or certify systems.
  3. Understand how state-of-the-art tools and frameworks for automated reasoning are used to build reliable and secure systems.

Part I: Security of Blockchains

  • We will cover existing blockchains (e.g., Ethereum, Bitcoin), how they work, what the core security issues are, and how these have led to massive financial losses.
  • We will discuss the state-of-the-art security tools (e.g., Securify) for ensuring that smart contracts are free of security vulnerabilities.
  • We will study the latest automated reasoning systems (e.g., Dagger) for checking custom (temporal) properties of smart contracts and illustrate their operation on real-world use cases.
  • We will study the underlying methods for automated reasoning and testing (e.g., abstract interpretation, symbolic execution, fuzzing) are used to build such tools.

Part II: Data Privacy

  • We will study how cryptographic primitives such as zkSNARKs are used to enforce data privacy for smart contracts deployed on public blockchains.
  • We will show how probabilistic solvers (e.g. PSI: PSI Solver) can be used to test and verify the differential privacy of probabilistically algorithms (e.g. data processing algorithms).
  • We will study how state-of-the-art program synthesis techniques are used automatically transforms programs and data queries to enforce privacy policies.

Part III: Machine Learning for Security

  • We will discuss how machine learning models for structured prediction are used to address security tasks, including de-obfuscation of binaries (DeBIN), Android APKs (DeGuard) and JavaScript (JSNice).
  • We will study to use program abstractions in combination with machine learning techniques to learn useful security rules from large codebases.

    Part IV: Security of Datacenters and Networks

    • We will show how to ensure that datacenters and ISPs are secured using declarative reasoning methods (e.g., Datalog). We will also cover the intersection of automated reasoning and software-defined networks such as P4.
    • We will discuss how to automatically synthesize secure configurations which lead to desirable behaviors, thus automating the job of the network operator and avoiding critical errors.
    • We will discuss how to apply modern discrete probabilistic inference so to reason about probabilistic network properties (e.g., probability of a packet reaching a destination if links fail).

    To gain a deeper understanding, the course will involve a hands-on programming project where the methods studied in the class will be applied.
