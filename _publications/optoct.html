Instructions

  • Virtual Machine
    1. Make sure you have 64-bit VirtualBox from Oracle.
    2. As some of the benchmarks take a lot of memory, make sure to give as much RAM as possible to the Virtual Machine (CRYPT benchmark in DPS Analyzer requires upto 12GB).
    3. The login credentials for the Virtual Machine are:
        username: pldi15
        password: pldi15
    4. Once the virtual Machine starts, go to the folder "pldi15-artifact" and open "Evaluation" file for instructions on how to compile the optoctagons library and run the analyzers.
    5. The virtual machine is the slower alternative as it does not support Intel's AVX Intrinsics. We measured upto 50% slowdown for optoctagons in virtual machine (running on the same machine). Some libraries used in Analyzers also run slowly in virtual machine so the speedup for end to end program analysis is also reduced.

  • Tar File (Preferably for 64-bit Linux)
    1. Extract the tar file.
    2. Go to the folder "pldi15-artifact" and open "Evaluation" file for instructions on how to compile the optoctagons library and run the analyzers.
